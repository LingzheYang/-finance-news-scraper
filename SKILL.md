## 功能特点

- **默认**: 获取最近1小时快讯 + 自动分析
- **单条分析**: 标题 -> 内容 -> 情绪/板块 -> 板块影响 -> 信号/风险
- **综合报告**: 市场情绪、板块详细影响分析、热门机会、风险提示、投资建议
- **JSON输出**: 保存完整数据，包含原文和分析结果

## 数据源

|---|---|---|
| 同花顺 | `--source tonghuashun` | HTTP API，支持分页 |

## 使用示例

### 默认（最近1小时 + 分析）

```bash
python scraper.py
```

### 指定时间段

```bash
python scraper.py --start 09:30 --end 11:30
```

### 只获取数据，不进行分析

```bash
python scraper.py --no-advice
```

### 保存到指定文件

```bash
python scraper.py --output my_news.json
```

**默认输出**: `latest.json`（最近运行的结果）

### 调整搜索页面数量

```bash
python scraper.py --pages 10
```

## 输出格式

**重要**: 显示所有快讯，不要省略任何一条。

### 单条快讯分析

每条快讯包含：
- **时间**: 快讯发布时间
- **标题**: 快讯标题
- **内容**: 完整的快讯正文
- **情绪**: 看多/看空/中性
- **板块**: 检测到的相关板块
- **板块影响**: 该事件对板块的影响方向（利好/利空/中性）和原因说明
- **信号**: 正面/负面关键词
- **风险**: 风险提示关键词

输出格式示例：
```
【1】10:02:12
【标题】现货黄金站上4430美元/盎司
【内容】现货黄金站上4430美元/盎司，日内涨1.1%...
【情绪】看多 | 【板块】黄金
【板块影响】黄金: 利好（避险需求推动金价上涨）
【信号】涨
```

### 综合分析报告

- **市场情绪统计**: 正面/负面/中性快讯数量
- **板块分析**: 各板块的影响方向和说明
- **热门机会**: 按板块分类的机会列表
- **风险提示**: 市场风险事件列表
- **投资建议**: 综合建议

## 常见问题

**Q: 为什么数据很少？**
A: 快讯数量与时间段相关，盘中数据多，盘前盘后数据少

**Q: 请求超时？**
A: 检查网络连接，或增加 `--pages` 参数尝试更多页面

**Q: 如何查看完整输出？**
A: 数据默认保存到JSON文件，可用 `python scraper.py --output result.json` 指定文件名
